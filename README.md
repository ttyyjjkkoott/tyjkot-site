# Minimal Blog

A clean, intuitive personal site built with React, Vite, and Supabase.

## Features

### Core Functionality
- **Feed Page**: Clean feed of recent posts, sorted by date
- **Search**: Full-text search across all stories
- **Menu Navigation**: Slide-out menu with smooth animations
- **Modular Pages**: About, Contact, Documents, Gallery, Tools
- **Admin Features** (login required):
  - Stories management (create, edit, delete)
  - Settings page with accent color customization
  - Protected routes

### Design Principles
- Minimal white background
- Black text with customizable accent color
- Rounded corners throughout
- Smooth animations and transitions
- Mobile responsive
- Intuitive navigation

### Technology Stack
- **React 18**: Component-based UI
- **Vite**: Fast build tool and dev server
- **Supabase**: Simple database and authentication
- **Lucide React**: Beautiful icon set
